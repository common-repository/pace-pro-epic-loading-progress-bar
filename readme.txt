== Description ==

Adds PACE page loading progress bar to your website, with live preview at the admin panel and dynamic color changing. And soon custom styles for each page and post will be added.

Original PACE JS Code by HubSpot: https://github.com/HubSpot/pace/

== Installation ==

1. Unzip the file
2. Upload it to wp-content/plugins
3. Activate it from the plugins section
4. You can change the settings from Appearance -> PACE Pro
